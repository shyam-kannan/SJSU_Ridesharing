# User Service

## Purpose

The User Service manages all user-related data and operations:

- User profile management (create, read, update, delete)
- Profile information (name, photo, bio, preferences)
- User preferences and settings
- Vehicle information for drivers
- User verification and ratings
- Document management (driver's license, insurance)
- Privacy settings
- User search and filtering

## Technology

- Node.js/TypeScript or Python
- PostgreSQL for user data storage
- S3 for document and photo storage

## Key Features

- Complete user profile management
- Driver and passenger role handling
- Vehicle registration and management
- User verification workflow
- Rating and review system integration
- Profile photo and document upload
